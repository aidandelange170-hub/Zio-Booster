# Zio-Booster Executable

This folder contains the Zio-Booster application as a standalone executable file.

## How to Run

### On Linux/Mac:
1. Open a terminal
2. Navigate to this folder: `cd /path/to/exe`
3. Run the executable: `./ZioBooster`

### On Windows:
1. Double-click the `ZioBooster.exe` file to run it

## About

This executable contains the complete Zio-Booster application with all necessary dependencies bundled. You don't need to install Python or any additional libraries to run it.

The application provides a graphical interface for launching HTTP requests against a target URL with configurable parameters like threads, duration, user-agent, and proxy settings.
