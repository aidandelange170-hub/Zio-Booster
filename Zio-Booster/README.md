# Zio-Booster FPS Booster

A modern FPS booster application that optimizes your system in the background to increase frame rates and reduce temperature by terminating high-temperature applications.

## Features
- Boosts frame rates in the background
- Reduces system temperature
- Terminates high-temperature applications
- Modern UI interface
- Automatic optimization
- Real-time monitoring

## Installation
1. Clone the repository
2. Run the start-application file to set up and run the application
3. Enjoy improved FPS performance!

## Requirements
- Python 3.7+
- psutil library
- tkinter for UI
